# LangChain + Gemini Intro Project

This is a beginner-level LLM project demonstrating how to use **LangChain with Google Gemini models** to build a simple chat-based AI assistant.

---

## 🚀 Features

- Uses Google Gemini 2.5 Flash model
- LangChain message-based structure
- System + Human message support
- Simple CLI-based interaction
- Beginner-friendly LLM integration demo

---

## 🧠 What This Project Does

This project:
- Sends a user question to a Gemini LLM via LangChain
- Uses system prompts to guide behavior
- Returns AI-generated responses

Example:
> Input: What is RAG?  
> Output: Explanation generated by Gemini model

---

## 🏗️ Tech Stack

- Python
- LangChain
- Google Gemini API
- Google Generative AI SDK

---

## 📂 Project Structure


app.py → Main application
requirements.txt → Dependencies
.env.example → API key template
notebook/ → Original Colab notebook


---

## ⚙️ Setup Instructions

### 1. Clone repo

git clone https://github.com/your-username/langchain-gemini-rag-intro.git
cd langchain-gemini-rag-intro


### 2. Install dependencies

pip install -r requirements.txt


### 3. Add API Key
Create `.env` file:

GEMINI_API_KEY=your_api_key_here


### 4. Run project

python app.py
